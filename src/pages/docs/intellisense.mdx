[Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) 通过为 Visual Studio Code 用户提供自动完成、语法高亮和语法检查等高级功能，提升了 Tailwind 开发体验。

<!-- - **Autocomplete**. Intelligent suggestions for class names, as well as [CSS functions and directives](/docs/functions-and-directives).
- **Linting**. Highlights errors and potential bugs in both your CSS and your markup.
- **Hover Previews**. See the complete CSS for a Tailwind class name by hovering over it.

- **自动补全**：智能提示类名，以及 CSS 函数和指令。
- **语法检查**：突出显示 CSS 和 HTML 中的错误和潜在 bugs。
- **悬停预览**：通过悬停在其上，可以看到一个 Tailwind 类名的完整的 CSS。
- **语法高亮显示**：提供语法定义，以便正确突出显示 “Tailwind” 功能。
